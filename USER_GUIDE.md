# User Guide: Sapian Cemetery Navigator

Welcome to the Sapian Cemetery Navigator! This guide will help you use the application to find and learn about graves in the cemetery.

## Getting Started

### Signing In

To use the application, you must first sign in. If you don't have an account, you will be prompted to create one.

1.  When you first open the application, you will be on the **Sign In** page.
2.  Enter your credentials to log in.

## Using the Application

Once you are logged in, you will see the main page of the application, which includes an interactive map of the cemetery.

### Main Features

*   **Interactive Map**: The main feature of the application is a map of the cemetery. You can pan and zoom to explore the cemetery grounds.
*   **Search for Graves**: Use the search bar to find a specific grave by name.
*   **Enable Location**: The application can use your device's location to provide walking directions to a selected grave. Click the **Enable Location** button to activate this feature.

### How to Find a Grave

There are two ways to find a grave:

1.  **Using the Search Bar**:
    *   Type the name of the person you are looking for into the search bar.
    *   As you type, a list of matching names will appear.
    *   Select a name from the list to see the location of the grave on the map and view more details.

### Navigation and Routing

Once you have selected a grave, you can get walking directions to it from your current location.

1.  **Enable Location**: First, make sure you have clicked the **Enable Location** button. This allows the application to know your starting point.
2.  **Select a Grave**: Find and select the grave you wish to visit.
3.  **View the Route**: The application will automatically display a green line on the map, showing the most direct walking path from your current location to the selected grave.

Follow the path on the map to reach your destination.

### Viewing Grave Details

When you select a grave, you will see the following information:

*   Name of the deceased
*   Age at the time of death
*   Date of birth
*   Date of death
*   Additional information
*   A picture of the grave, if available

## For Administrators

If you have an "admin" account, you will see an "Administrator" button that will take you to the admin dashboard.

### Admin Features

From the admin dashboard, you can:

*   Draw new grave lots directly on the map.
*   Add new graves and assign them to lots.
*   Edit the details of existing graves.
*   Upload images for graves.
*   View a complete list of all graves.
*   Search for graves using various criteria.
